    Курс по Java Web Development
    Бесплатный тренинг Java Web Development - это введение в разработку программного обеспечения на языке программирования Java и связанных с ним технологиях. Курс предназначен для кандидатов с небольшим опытом программирования, участники тренинга познакомятся с основами Java и Java Web.

    Базовый курс по английскому языку
    Общий курс английского языка – это иностранный язык «для жизни». Программа обучения в Гомеле доступна на уровнях подготовки от Starter («с нуля») до Upper Intermediate (B2) и направлена на практическое овладение английским языком в комплексе и преодоление психологического барьера при общении.
